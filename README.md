# Quiz CRUD System
A complete CRUD (Create, Read, Update, Delete) system for managing quiz questions, developed in Python with Tkinter GUI as a technical course project at SENAC.

# About the Project
This project implements a full-featured quiz management system with a graphical user interface. Questions are stored in JSON format and can be easily created, edited, viewed, and deleted through an intuitive interface.
- Developed by: Marcela & Giovanna
- Institution: SENAC - Technical Course in Systems Development
- Purpose: Git/GitHub Integration Project

# Features

- Create new questions with 3 answer options
- Read and display all registered questions
- Update existing questions
- Delete questions from database
- User-friendly GUI built with Tkinter
- Data persistence using JSON
- Input validation for all fields

# Technologies Used

Python 3.x
Tkinter - Graphical User Interface
JSON - Data Storage
Git/GitHub - Version Control
